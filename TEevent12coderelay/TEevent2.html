Code Relay Race

Code Relay Race is a fun and competitive programming event where participants work together in teams to solve a problem, but with a twist: each team member writes only part of the solution and then passes it to the next member to continue.

Rules:

1. Team Formation: Teams consist of 3-5 members.

2. Time Limit: Each team has a total time (e.g., 45 minutes) to complete the challenge, with each member getting a set time (e.g., 10 minutes) before switching to the next member.

3. Problem Complexity: The problem should require multiple steps or modules to complete (e.g., a multi-function program or a sequence of tasks).

4. Relay Mechanism: After each member’s time is up, they must stop immediately, pass the code to the next member, and provide a brief explanation (in less than 1 minute).

5. No Backtracking: Team members can only build upon the previous member's code— they cannot delete or significantly alter it.

6. Scoring Criteria: Points awarded based on code functionality, team collaboration, and clean coding practices.
