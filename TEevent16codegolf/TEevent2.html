Code Golf (Minimize Code Size)

Code Golf (Minimize Code Size) is a fun and challenging event where the goal is to solve a programming problem using the smallest possible amount of code. Participants must balance functionality with code brevity, using creative and concise programming techniques.

Rules:

1. Individual or Team Participation: Participants can compete individually or in teams of 2.

2. Problem Statement: Each participant/team is given a specific problem to solve (e.g., simple algorithms, string manipulations, or mathematical functions).

3. Language Choice: Participants can choose from a list of programming languages (e.g., Python, JavaScript, C), with some languages having character count adjustments based on verbosity.

4. Time Limit: Participants are given a fixed time (e.g., 30-60 minutes) to solve the problem and optimize the code size.

6. Tie-Breaker: If two or more participants have the same code length, tie-breaking criteria may include execution time or creative approach.

7. Scoring Criteria: * Code Size: Points are awarded based on how few characters the solution uses. *Functionality: The code must solve the problem completely and correctly. Incorrect solutions, even if short, receive zero points.
